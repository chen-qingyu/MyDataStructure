Detailed Description

template<typename T>
class mdspp::Set< T >::Iterator

Set iterator class.

Walk the set in ascending order. This means that begin() is the smallest element.

Because the internal elements of the set have a fixed order, thus the iterator of the set only supports access and does not support modification.
