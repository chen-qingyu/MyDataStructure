Detailed Description

template<typename K, typename V>
class mdspp::Map< K, V >::Iterator

Map iterator class.

Walk the map in ascending order. This means that begin() is the smallest pair.

Because the internal pairs of the map have a fixed order, thus the iterator of the map only supports access and does not support modification.
